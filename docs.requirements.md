# Requirements

## Stakeholders
- University administrators
- Student welfare teams
- Academic departments

## Key Questions
- Do schedules (morning vs evening classes) affect stress and GPA?
- How do campus environments (noise, crowd density) impact well-being?
- Which services (counseling, library, fitness) correlate with productivity?

## Data Needed
- Student demographics
- Course enrollments
- Class schedules
- Campus environment readings
- Service usage logs
- Well-being surveys
- Academic outcomes (GPA, attendance)
